# Kocer Doc System

## التشغيل

1. افتح الطرفية في المجلد: `cd /workspaces/Kocer/kocer-doc-system`
2. شغّل الخادم الخلفي: `npm run server`
3. شغّل الواجهة الأمامية: `npm run dev`
4. افتح المتصفح على: http://localhost:3000

## الرموز المتاحة

- ABC123XYZ → شركة النخبة للتجارة العامة
- DEF456UVW → شركة الرافدين للمواد الغذائية
- GHI789RST → شركة دجلة للإلكترونيات

## ملاحظات

- الخادم يعتمد على ملف البيانات الموجود في `server/data/db.json`
- يتم ربط الواجهة بالـ API عبر `/api`
